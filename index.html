Генератор диалогов

Инструкции

NPCName: Имя Вашего НПС

Id: уникальное названия блока диалога, для перехода между ними, если это начальный/первый диалог, напишите тут Start

initialPhrase: фраза/ответ нпс

ResponseTextKey: фраза которую может сказать игрок(его ответ)

NextDialogueID: тут надо указать Id диалога который будет запускаться при нажатии на этот ответ

TriggerKey: Важная часть, нужнно для запоминания диалогов или изменения его вида, вот примеры:

  • Me - диалог видеться от лица игрока, с его портретом
  • Thoughts - мысли игрока, без портрета и имени
  • ProblemMentioned - запускает уникальный диалог когда ты сказал о проблемах
Если вам не надо такое, оставьте пустым

Уникальные замениики в initialPhrase:

  • [pX] - где X это число, нужно чтобы сменить портрет X это индекс, начало от 0, это первый, пример: [p1]
  • [iX] где X это число, нужно чтобы выдать предмет игроку, X это айди предмета начало от 0, это первый предмет, пример [i0]
  • [c] - это заменяеться на имя кота которое указал игрок в игре.

Как пользоваться:

Для начала введите Имя НПСи нажмите добавить диалог, после чего введите Start так как это начало, фразу которую вам надо, и нажмите добавить ответ игрока, если он может ответить, если ненадо не нажимайте. потом как нажали введите данные где NextDialogueID надо указать именем которым назовёте следуйщий диалог, то есть будет Test допустим, потом нажмите снова добавить добавить диалог если вы всё заполнили и вам надо столько ответов, и укажите в id Test (название какое было в NextDialogueID ) и делайте столько раз сколько вам надо
